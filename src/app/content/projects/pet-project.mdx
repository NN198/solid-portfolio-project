## Features 

- **User Authentication**: Allow users to sign up and log in.
- **Add Pets**: Allows users to register their pets and store it in the database
- **Booking Appointments**: Displays the available slots for appointments and validates booking accordingly.
- **Appointment History**: View appointment history and track upcoming bookings.

<br/>
<br/>
## Technologies

- **ReactJS**: React was used for client-side building static and dynamic websites.
- **Maven**: A build automation tool primarily used for Java-based projects. 
- **Spring Boot Framework**: A powerful Java framework for creating microservices and standalone applications with minimal configuration.
- **TomCat**: An open-source implementation of the Java Servlet, JavaServer Pages, and WebSocket technologies. 
- **SQLite**: A lightweight, serverless, self-contained database engine

## Getting Started

To get started with this project, you can clone the repository and install the
dependencies (spring dependencies were added to `pom.xml`):


```bash
git clone
cd PetPortal
```
Change the `application.properties` in the Maven Project for seamless connection to your database
```bash
spring.datasource.driver.class=com.mysql.cj.jdbc.Driver
spring.datasource.url=jdbc:mysql://<database name>
spring.datasource.username=<username>
spring.datasource.password=<password>
spring.jpa.properties.hibernate.dialect=org.hibernate.dialect.MySQL5Dialect

## Show sql query.
spring.jpa.show-sql=true

## Hibernate ddl auto.
spring.jpa.hibernate.ddl-auto=create
server.port = 8080 <As per the spring boot server configuration>
```
<br/>
<br/>
Run the application under the name `"ClinicpetApplication.java"` that will contain the runnable interface of Spring (@SpringBootApplication)
<br/>

The backend server will start on TomCat at `http://localhost:8080`
<br/>

To install node dependencies for PetPortalFrontEnd using the below command:

```bash
cd PetPortalFrontEnd
npm install 
```
or 
```
npm i
```




Once the dependencies are installed, you can run the development server:

```bash
npm run dev
```

The development server will start at `http://localhost:3000`.
