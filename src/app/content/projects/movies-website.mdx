## Features ##

- **Home Page**: Displays all the movies from latest to oldest along with header to navigate to different pages
- **Top-rated/Upcoming Recommendations**: Give recommendations as per the ratings for maximum users
- **Search Movies/Actors**: Allows users to search from a myriad of movies using the Search API
- **Ratings**: Displays the ratings scrapped from IMdB and its subsidiaries

## Getting Started

To get started with this project, you can clone the repository and install the
dependencies:

To install npm modules dependencies for Angular make sure to have it installed previously using the below command:

```bash
cd ecommerce-product
npm install 
```
or 
```
npm i
```
or 
```
ng add .
```
This application will run on Angular 11, to avoid conflict issues makes sure Node version is 12 or below.

Once the dependencies are installed, you can run the development server:

```bash
npm run dev
```
or

```bash
ng serve
```

The development server will start at `http://localhost:4200`.

## Deployment

To deploy this project, you can use Vercel. You can link your GitHub repository
to Vercel and deploy the project with a single click.
To avoid linting or typescript conflicts during deployment save this in ~`tsconfig.app.json` 

```
eslint: {
    // Warning: This allows production builds to successfully complete even if
    // your project has ESLint errors.
    ignoreDuringBuilds: true,
  },
  typescript: {
    // !! WARN !!
    // Dangerously allow production builds to successfully complete even if
    // your project has type errors.
    // !! WARN !!
    ignoreBuildErrors: true,
  },
```
