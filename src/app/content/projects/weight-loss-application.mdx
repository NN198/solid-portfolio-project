## Features 

- **User Login Page**: Allow users to login and gather basic information of their weight, bmi, sleep cycle and groceries they visit
- **Geolocation-Based Recommendations**: Based on the initial location set by the user the app sends notifications to alert the users to buy groceries or meals that would nourish their diet
- **Pedometer/Accelerator Sensors**: Gather data the steps covered by the user during the day or their work outs
- **Notification/Alert Settings**: These notification were tailored to motivate the user to stay on track with their fitness goals and maintain an active lifestyle. Alerts to show the step count of the user especially when they are close to fitness studios or grocery stores

## Getting Started

To get started with this project, you can clone the repository.
In Android Studio, go to the top-right corner and click Sync Project with Gradle Files.
This will download all necessary dependencies specified in your `build.gradle.kts` files.


If you are using command line to run the project 
- Open a terminal or command prompt.
- Navigate to the project's root directory (where gradlew is located).
-Run the following command to build:
```bash
git clone
cd weight-loss-application
./gradlew build
```


To install dependencies and build an Android project in Android Studio, you can use Gradle, the default build tool for Android projects. Follow these steps to set up and build your project:
Navigate to the project's root directory (where gradlew is located).
<br/>
Run the following command to build:

```bash
./gradlew build
```
<br/>

This will download dependencies, compile the code, package the app into APKs.
Once the dependencies are installed, you can run the build through Android Studio IDE or through 
Click Run -> Run 'app' in Android Studio, or execute:
```bash
./gradlew installDebug
```
